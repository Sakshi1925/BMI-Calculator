# BMI-Calculator

• Project Title

BMI Calculator


• Overview of the Project

1.	A simple application that calculates Body Mass Index (BMI).
2. Takes user inputs such as height and weight.
3. Displays BMI value along with the health category (Underweight, Normal, Overweight, Obese).


• Features

1.User-friendly input form for height and weight.
2. Input validation (checks empty or invalid values).
3. Automatic BMI calculation using the standard formula.
4. Clear output display with interpretation.
5. Can be implemented as console, GUI, or web version (depending on your choice).


• Technologies/Tools Used

1.	Programming Language: Python
2.  Optional tools: IDE - VS Code
                    Git & GitHub for version control



• Steps to Install & Run the Project

1. Clone or download the GitHub repository.
2. Open the project folder in your preferred IDE.
3. Ensure the required language runtime is installed.
4. Run the main program file: Python - Source Code.py
5. Follow the on-screen instructions to input height and weight.



• Instructions for Testing

1.	Test with different height and weight values.
2. Use valid numeric inputs to ensure accurate calculation.
3. Compare results with an online BMI calculator for validation.
